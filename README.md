## ✨ Key Features
- 🧭 **Structured multi-page portfolio** with homepage, about, projects, blogs, notes, and contact pages.
- 🌗 **Theme switching** with dark and light modes using CSS variables.
- 🪟 **Card-based UI** with gradients, borders, and soft shadows for depth.
- 📱 **Responsive layout** optimized for desktop, tablet, and mobile screens.
- 🧩 **Interactive project showcase** with a dedicated Nexus AI detail page and gallery.
- 🧷 **Consistent branding** across all pages with a reusable header and navigation system.
- 📄 **Document links** for CV and resume access from the homepage.

## 🎨 Design Language

The website uses a compact visual system built around a few clear ideas:

1. **Editorial spacing** - content is grouped into clean blocks with strong visual rhythm.
2. **Layered surfaces** - cards, panels, and buttons use borders, shadows, and subtle gradients.
3. **Theme consistency** - all pages use the same light/dark variables and typography rules.
4. **Responsive grids** - layouts collapse smoothly on smaller screens without breaking hierarchy.
5. **Brand consistency** - the same header branding, icon style, and color language appear throughout the site.

## ⚙️ Run Locally

### 1. Open the project
Clone or download this repository and open the folder in VS Code or any editor.

### 2. Launch the website

The README image previews depend on the `assets/` folder structure, so keep the directories unchanged.
